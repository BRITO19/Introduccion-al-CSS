ACTIVIDAD 1 CSS

  1. ¿Qué es CSS?

    2. CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para describir la presentación de documentos HTML. Permite definir cómo se mostrarán los elementos en la pantalla, en papel, o en otros medios.

  2. ¿Cuál es la función principal de CSS?

    3. La función principal de CSS es separar el contenido (HTML) de la presentación. Esto incluye la configuración de colores, fuentes, espacios y diseño general de las páginas web.

  3. ¿Cómo se puede incluir CSS en un documento HTML?

    4. Se puede incluir de tres maneras:

    • Inline: Usando el atributo style dentro de una etiqueta HTML.
    • Internal: Colocando el CSS dentro de una etiqueta <"style"> en la sección del documento HTML.
    • External: Enlazando a un archivo CSS externo con la etiqueta en la sección .
  4. ¿Qué son los selectores en CSS?

    5. Los selectores son patrones que se utilizan para seleccionar los elementos HTML que se desean estilizar. Ejemplos incluyen selectores de tipo (como p), de clase (como.clase), de ID (como #id), y combinaciones de estos

  5. ¿Qué son las propiedades y valores en CSS?

    6. Las propiedades son atributos que se pueden aplicar a los selectores (como color, font-size, margin). Los valores son las configuraciones específicas que se asignan a esas propiedades (como red, 16px, 10px).

  6. ¿Cómo se aplica el modelo de caja en CSS?

    7. El modelo de caja es un concepto que describe cómo se representan los elementos en CSS. Cada elemento se considera una caja con márgenes, bordes, relleno (padding) y contenido. Este modelo determina el espacio que ocupa un elemento y cómo se relaciona con otros elementos.

  7. ¿Qué es el "cascading" en CSS?

    8. El "cascading" se refiere a cómo se aplican las reglas de CSS en orden de prioridad. Cuando hay múltiples reglas que afectan a un mismo elemento, el navegador determina cuál aplicar basado en la especificidad, el orden de las reglas, y si son importantes (uso de !important).

  8. ¿Qué son las media queries?

    9. Las media queries son técnicas que permiten aplicar estilos CSS específicos basados en características del dispositivo, como el tamaño de la pantalla o la resolución. Se utilizan para crear diseños responsivos que se adaptan a diferentes dispositivos.

  9. ¿Cuáles son algunos estilos comunes que se pueden aplicar?

    10. Algunos estilos comunes incluyen:

    • Color de texto (color)
    • Tamaño de fuente (font-size)
    • Espaciado (margin, padding)
    • Bordes (border)
    • Fondo (background-color, background-image)
  10. ¿Cómo se puede usar CSS para crear un diseño responsivo?

    11. Para crear un diseño responsivo, se pueden usar técnicas como:

    • Flexbox y Grid para crear layouts flexibles
    • Media queries para aplicar estilos diferentes en función del tamaño de la pantalla
    • Unidades relativas (como %, em, o rem) en lugar de unidades fijas (px) para que los elementos se ajusten mejor a diferentes tamaños de pantalla.
